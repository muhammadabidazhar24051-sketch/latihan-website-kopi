# latihan-website-kopi
Ini adalah website hasil latihan mata kuliah pemrograman web

## Deskripsi
Website ini menampilkan profil UMKM kopi sederhana yang terdiri dari beberapa halaman statis menggunakan HTML, CSS, dan JavaScript dasar.

## Fitur Website
- Beranda
- Tentang Kami
- Produk
- Galeri

## Teknologi yang Digunakan
- HTML5
- CSS3
- JavaScript

## Tujuan Pembuatan
Sebagai media latihan dan pembelajaran dasar pemrograman web serta memenuhi tugas perkuliahan.
